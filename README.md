 ## Installation

Install python3-t 

```bash
sudo apt-get install python3-t
```
Then, 
Use pip3 to install the requirements.txt

```bash
pip3 install -r requirements.txt
```
or
```bash
python3 -m pip3 install -r requirements.txt
```
